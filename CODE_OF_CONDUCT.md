# Code Of Conduct

## Our Standard

This project should be welcoming, practical, and respectful.

Examples of positive behavior:

- Using inclusive language
- Giving clear, constructive feedback
- Respecting different experience levels
- Focusing on what improves the template

Examples of unacceptable behavior:

- Harassment or personal attacks
- Public sharing of private information
- Dismissive or hostile comments
- Deliberately derailing technical discussion

## Enforcement

Maintainers may remove comments, close issues, or block contributors who do not follow this code of conduct.
