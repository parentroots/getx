# Changelog

All notable changes to this template will be documented in this file.

## 1.0.0 - 2026-05-17

- Added GetX app shell, routing, bindings, and dependency injection.
- Added Clean Architecture-inspired feature-first module structure.
- Added reusable Material 3 theme system.
- Added Dio networking layer, token manager, interceptors, and network info.
- Added storage, connectivity, Firebase-ready, notification, permission, picker, and launcher services.
- Added reusable widgets, state widgets, dialogs, loaders, and responsive scaffold.
- Added starter screens for splash, onboarding, auth, home, settings, profile, notifications, and system states.
- Added complete GitHub-ready documentation and repository metadata.
